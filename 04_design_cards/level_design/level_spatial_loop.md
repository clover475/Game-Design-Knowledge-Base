# 一句话定义

让玩家通过绕行、解锁捷径或重新进入旧空间，形成对地点的记忆和掌控感。

# 玩家动作

探索、发现锁门、绕行、获得钥匙或信息、打开捷径、回到熟悉地点。

# 系统规则

空间回环需要有明确入口、障碍、绕行路径、解锁条件和回报。

# 反馈方式

门打开、灯光变化、NPC 位置变化、地图标记更新。

# 可迁移到当前项目的方式

犍陀罗文化游戏中，可以把寺院、石窟、集市和遗址设计成多层回环空间，让文化信息随路径逐步揭示。
