# Priority Reading List

## A 级：第一阶段必须读

1. MDA: Mechanics, Dynamics, Aesthetics
   - 目标：建立“机制 -> 动态 -> 体验”的基本分析框架。
   - 输出：1 份 AI 阅读笔记，3 张机制卡。

2. The Level Design Book
   - 目标：理解 blockout、动线、视线引导、空间节奏、环境叙事。
   - 输出：1 份关卡设计 checklist，3 张关卡卡。

3. Game Programming Patterns
   - 目标：理解策划需求如何落到开发结构上。
   - 输出：开发需求模板，技术实现卡。

4. Emily Short Interactive Storytelling
   - 目标：理解互动叙事、对话系统、玩家表达。
   - 输出：叙事卡、NPC 对话系统问题清单。

5. Game Mechanic Explorer
   - 目标：把抽象机制变成可交互原型。
   - 输出：机制到前端交互需求模板。
