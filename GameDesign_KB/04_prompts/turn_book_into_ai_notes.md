# Turn Book Into AI Notes

请把以下书籍内容整理成可被 AI 调用的 notes。

要求：

- 不要复述全文
- 提炼概念、方法、框架、案例
- 每条 note 都要能用于后续设计任务
- 标记适用主题：game_design / mechanics / level_design / narrative / visual / development

输出格式：

```markdown
---
title:
type: ai_note
topic:
source_url:
local_path:
status: draft
tags:
---

# Core Ideas

# Design Methods

# Useful Examples

# How AI Should Use This
```
