# GameDesign KB

这是一个可直接用 Obsidian 打开的游戏设计知识库 Vault。

## 目录

```text
GameDesign_KB/
├── Home.md
├── 00_books/
├── 01_open_resources/
│   ├── github_repos/
│   ├── open_books/
│   ├── articles/
│   ├── gdd_examples/
│   └── postmortems/
├── 02_indexes/
│   ├── book_index.md
│   ├── repo_index.md
│   ├── missing_books.md
│   └── download_log.md
├── 03_ai_notes/
├── 04_prompts/
└── README.md
```

## 使用方式

1. 用 Obsidian 打开 `GameDesign_KB/` 文件夹。
2. 打开 `Home.md`。
3. 先读 `02_indexes/repo_index.md`，了解本地已有公开 repo。
4. 你手动准备书籍后，把文件夹交给 Codex 导入 `00_books/`。
5. 用 `04_prompts/` 中的 4 个 prompt 把书、repo、文章转成 AI notes。

## 当前策略

- 公开 GitHub repo 已 clone 到本地，但不提交到主 GitHub 仓库，避免嵌套仓库和体积问题。
- 书籍区暂时留空，等用户提供本地文件后再导入。
- 商业书先记录到 `missing_books.md`，并给出开放替代资料。
