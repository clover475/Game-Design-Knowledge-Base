# 项目一句话

以犍陀罗文化为核心的探索叙事游戏，把历史、宗教、艺术、地理和人物记忆转化为可探索、可修复、可对话的系统。

# 目标体验

探索、发现、修复、误读与再理解。玩家不是被动读资料，而是通过行动理解文化线索。

# 核心循环

探索场景 -> 收集线索/碎片 -> 与 NPC 或文物对话 -> 解释或修复 -> 解锁新路径/新叙事层 -> 形成新的文化理解。
